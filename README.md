# College Hackathon Attendance

This project is a College Hackathon Attendance system built with a React frontend and a Node.js backend.

## Structure

- **client**: React frontend application.
- **server**: Node.js backend server.
- **client-html**: Static HTML files (if applicable).

## Getting Started

1.  Navigate to the `server` directory and run `npm install` then `npm start`.
2.  Navigate to the `client` directory and run `npm install` then `npm run dev`.
